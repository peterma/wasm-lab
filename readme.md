## Java

Install JDK; 

Add environment variable: JAVA_HOME;

Add %JAVA_HOME%\bin to PATH;


Install Maven; 

Add env variable: MAVEN_HOME;

Add %MAVEN_HOME%\bin to PATH;


javac SquareCalculator.java
